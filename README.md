# Get started

## Installation

1. Create a virtualenv with Python 3.8
2. `pip install -r dev_requirements.txt`

## Plot clusters with streamlit
To look at clusters run 
```bash
PYTHONPATH=. streamlit run notebooks/st_plot_clusters.py
```
